## üìñ About The Project

**Space Game** is a minimalist STG (Shooting Game) engine built for the web. It delivers classic arcade shooter action with modern performance. Players can enjoy infinite progression with no dependencies, making it easy to run in any browser. The game features a custom entity pooling system for smooth performance, allowing you to shoot hundreds of bullets without a hitch.

### ‚ú® Key Features

* **Infinite Scaling:** Your weapon levels have no limits. Firepower increases as you survive longer.
* **Procedural Bullet Patterns:** Enjoy varied projectile spread that adapts to your weapon level.
* **Precision Control:** Navigate with keyboard controls and utilize "Focus Mode" (press `Shift`) for sharp dodging.
* **High Performance:** Uses Object Pooling and Delta-Time physics, ensuring consistent gameplay without pauses.
